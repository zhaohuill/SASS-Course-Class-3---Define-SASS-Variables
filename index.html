Define SASS Variables:_

In SASS you can Define Variables Just Like in JavaScript; this is to facilitate the Repeated Use of certain Values that are difficult to memorize (Like Hexadecimal Colors, Sizes in Different Measures, etc.) in Large Scale Projects.
It's very easy to do; you just have to declare the variable as you would in Javascript BUT with a "$" Sign at the beginning of the Variable Name, assign the value you want to that variable and finally put the Variable Name with the "$" Sign included where you want that value you assigned to be.
You can see real examples in the File "styles.scss" of this Repository.

You can Learn More by Reading the SASS Official Documentation.
